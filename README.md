Lightweight whiteboard plugin for IDA Pro that allows reverse engineers to sketch and brainstorm directly inside IDA.

## Features
- Text
- Drawing
- Eraser
- Selection
- Undo / redo
- Styles
- Clear canvas

## Requirements
PyQt5 must be installed before using DrawIDA.<br>
`pip install PyQt5`

## Installation
Copy `DrawIDA.py` to your IDA plugins folder and restart IDA.

## Compatibility
- IDA 7.4 → IDA 9.x
